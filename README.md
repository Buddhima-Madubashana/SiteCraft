# SiteCraft - Premier Commercial Construction

SiteCraft is a modern, premium landing page designed for a commercial construction business based in Colombo. Built with a focus on high-quality visuals, a dark-themed aesthetic with vibrant amber/orange accents, and a user-friendly layout.

## 🚀 Features

*   **Hero Section:** Striking first impression with left-aligned typography, clear value proposition, and a modern construction background.
*   **Trust Indicators:** Displays licenses, bonding, and safety compliance to establish immediate credibility.
*   **Core Services:** Scannable breakdown of main pillars (New Builds, Renovations, Roofing, Commercial Fit-Outs).
*   **Project Portfolio:** Visual gallery of completed projects with an interactive Before/After image comparison slider effect for renovations.
*   **Social Proof:** Authentic client testimonials featuring 5-star reviews.
*   **About Section:** Humanizes the brand summarizing its 20+ years of adherence to deadlines and safety.
*   **Lead Capture:** Frictionless contact form designed for high conversion.
*   **Responsive Footer:** Professional layout with navigation, social links, and legal information.

## 🛠️ Technology Stack

*   **Framework:** [React 19](https://react.dev/)
*   **Build Tool:** [Vite](https://vitejs.dev/)
*   **Language:** [TypeScript](https://www.typescriptlang.org/)
*   **Styling:** Pure Vanilla CSS (No external UI frameworks) - fully custom design system using CSS variables.
*   **Icons:** [Lucide React](https://lucide.dev/)
*   **Fonts:** Outfit (Google Fonts)

## 💻 Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository or extract the project files.
2. Navigate into the project directory:
   ```bash
   cd SiteCraft
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

### Running Locally

To start the local development server:

```bash
npm run dev
```

Open your browser and navigate to the URL provided in the terminal (usually `http://localhost:5173` or `http://localhost:5174`).

### Building for Production

To create a production-ready build:

```bash
npm run build
```

The compiled assets will be placed in the `dist` directory. You can preview the production build locally by running:

```bash
npm run preview
```

## 🎨 Design System

This project uses a custom CSS variables system defined in `src/index.css` for consistent theming. The primary palette consists of:
*   `--bg-dark`: Deep Slate blue (`#0f172a`)
*   `--primary`: Construction Amber (`#f59e0b`)

Global typography is powered by the **Outfit** font for a sleek, modern, and highly readable look.
